# Лицевая часть комплекса "Система управления организациями"

## О лицевой части

Лицевая часть комплекса предоставляет графический интерфейс для использования функциональности, предоставляемой сервером (подробнее см. [тут](https://github.com/pamugk/org-info-server.git "Ссылка на хранилище кода сервера")). Следовательно, в отсутствие доступа к серверу лицевая часть абсолютно бесполезна.

## Нюансы использования

Лицевая часть поделена на 6 основных экранов (3 типа экранов, каждый тип реализован для сотрудников и организаций): список сотрудников / организаций, дерево сотрудников / организаций, редактор (совмещающий создание и изменение) сотрудников / организаций.  
Экран создания сотрудника / организации может быть вызван только с экранов список сотрудников / организаций соответственно путём нажатия кнопки "Создать".  
Деревья организаций и сотрудников подгружаются по частям таким образом, что каждая часть дерева подгружается только при попытке пользователя просмотреть её (при этом пользователю предоставлена возможность обновить часть дерева без необходимости загружать его целиком заново). Для дерева сотрудников также реализован пейджинг дочерних элементов.
## Немного деталей реализации

Лицевая часть разработана с использованием ReactJS.  
Сборка осуществляется посредством Webpack.  
Графический интерфейс разработан с использованием компонентов MaterialUI.
